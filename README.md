# Student Helper Assistant

A lightweight web application designed to help students manage their daily academic tasks, track grades, and organize deadlines efficiently.

## Objective
The main purpose of this project is to create a centralized tool for academic productivity. iIt allows students to monitor their progress and stay organized throughout the semester.

## Techologies Used
HTML5
CSS3
JavaScript

## Features
**Task Scheduler:** Add and manage daily academic tasks.
**Deadline Tracker:** Keep track of upcoming project and exam deadlines.
**Grade Overview:** A simple tool to input test scores and calculate progress.

## Why this project?
This application was created to combine my passion for programming with my interest in academic organization. It served as a practical way to apply web development skills to solve real-world student challenges.
